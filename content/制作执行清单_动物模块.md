## 第三阶段：音频生成（1 天）

### 工具：Azure TTS 免费额度

1. 注册 Azure 账号
2. 创建 Speech Service 资源
3. 获取 Key 和 Region
4. 设置环境变量：
   ```bash
   export AZURE_SPEECH_KEY=your_key
   export AZURE_SPEECH_REGION=japaneast
   ```
5. 安装依赖：
   ```bash
   pip install azure-cognitiveservices-speech
   ```
6. 运行脚本：
   ```bash
   python scripts/generate_audio.py
   ```

### 生成文件

每个动物生成 3 个音频：

| 类型 | 文件名示例 | 内容 |
|---|---|---|
| 单词 | cat_word.mp3 | 猫，māo |
| 例句 | sentences/cat_sentence.mp3 | 这是一只猫。 |
| 叫声 | cat_sound.mp3 | 喵喵 |

### 备选工具

- 如果 Azure 不方便，可用 Google Cloud TTS（每月 400 万字符免费）
- 或 Amazon Polly（每月 500 万字符标准语音免费）
- 应急方案：自己用手机录音，用 Audacity 剪辑

## 第五阶段：互动页面原型（2-3 天）

### 工具：Genially 免费版

页面 1：开场
- 小豆打招呼：「你好！今天我们来认识动物。」
- 播放欢迎音频

页面 2-9：每个动物学习页
- 大图居中
- 点击图播放：「猫，māo」
- 点击「例句」按钮播放：「这是一只猫。」
- 小豆旁白：「小猫喵喵叫。」
- 「下一个」按钮

页面 10：听音找图
- 4 个动物图标
- 播放「猫，māo」
- 点对了：星星 + 鼓励音
- 点错了：小豆说「再试一次」

页面 11：结束
- 小豆：「你今天认识了 4 个动物，真棒！」
- 奖励动画
